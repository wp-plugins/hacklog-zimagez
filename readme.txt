== Description ==

Hacklog ZimageZ plugin is developed for users who uses ZimageZ.com ‘s images hosts.(free)

ZimageZ.com is a free image hosts website.
you can upload images to the site,and post it on your own wordpress blog.
With an ZimageZ account,you can upload and manage the images you’ve uploaded.
With Hacklog ZimageZ plugin,you can upload image in WordPress easily by click the “ZimageZ” button near the “media” button above your WordPress editor,and then, You can insert the image to your post.

== Installation ==

1. Upload the whole `hacklog-zimagez` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Settings`==>`Hacklog ZimageZ` to fill your loign name and password on zimagez.com
